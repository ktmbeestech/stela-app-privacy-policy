# Account Deletion Request

Users can permanently delete their account directly from the application.

## How to Delete Your Account

1. Open the application
2. Go to **Profile** then Go to **Settings**
3. Select **Delete Account**
4. Confirm the deletion request

Once confirmed, the following data will be permanently removed:

* Profile information
* Tasks and reminders
* Notification history
* Device tokens
* Saved preferences

## Data Retention

We do not retain deleted account data after account removal is completed, except where required for legal or security purposes.

## Need Help?

If you are unable to access your account or require assistance, please contact support through the application support section.
